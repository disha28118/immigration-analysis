# Placement Data Analysis

## Project Summary
The goal of this project is to analyze the student placement data.

## Tech Stack
- python
- pandas
- numpy
- matplotlib
- streamlit

## Installation
1. Clone the repository
```bash
git clone <repo-url>
```

2. Install the dependencies
```bash
pip install - requirements.txt
```
